# Monitor de Armazenamento e Memória

App Android profissional para monitorar armazenamento e memória RAM.

## Fase 1

- Mostra armazenamento total, usado e livre.
- Mostra memória RAM total, usada e livre.
- Atualiza a tela automaticamente a cada 30 segundos.
- Usa WorkManager para leitura periódica em segundo plano.
- Mostra notificação fixa.
- Não acessa arquivos pessoais.
- Não apaga arquivos.
- Base preparada para evoluir para scanner, limpeza manual segura e Google Play Store.
